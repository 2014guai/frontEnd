# 概述
* 前端是啥？简单的说，呈现页面，实现用户交互。    
* 前端需要会啥？html：结构，构建页面；css：表现，样式美化；js：行为，用户交互；其他：SEO优化、性能优化、脚本压缩、缓存控制等。
*一个设计优良的网页，应该做到结构、表现、行为三者分离*
