###闭包      
 
* 理解:
  * 通俗：
    * 当嵌套的内部函数引用了外部函数的变量时就产生了闭包
  * 个人：
    * 通过chrome工具得知: 闭包本质是内部函数中的一个对象, 这个对象中包含引用变量属性
* 作用:
  * 延长局部变量的生命周期
  * 让函数外部能操作内部的局部变量
* 写一个闭包程序
  function fn1() {
    var a = 2;
    function fn2() {
      a++;
      console.log(a);
    }
    return fn2;
  }
  var f = fn1();
  f();	//3
  f();	//4
* 闭包应用:
  * 回调函数
  * 模块化
  * 循环遍历加监听
  * jQuery
* 缺点:
  * 变量占用内存的时间可能会过长, 可能导致内存泄露
  * 解决:
    * 及时释放 : f = null;
